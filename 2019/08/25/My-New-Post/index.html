js简略

js的声明

  • 在head标签中使用script标签进行js代码域的声明

      • 作用：
        • 声明js代码域
      • 特点：js的代码只会作用于当前网页

  • 在head标签中使用script标签引入外部声明好的js文件

      • 作用：引入外部声明好的js文件
      • 特点：
        • 实现js代码的重复使用，避免代码的冗余。

  • 注意：

    • 因为js在HTML文档中是一门单独的语言，可以声明在文档中的任意位置。
    • 一般情况下声明在head标签中。

js的变量声明

  • js中的所有变量声明只有var关键字。

      • 注意：
        • js的变量名是严格区分大小写的。
        • js中的字符可以使用双引号也可以使用单引号。
        • js中可以声明同名变量，不会报错，后面的会将前面的声明覆盖。

  • js中的数据类型

    • 数据类型判断关键字：typeof
    • 数据类型：
      • number:所有的数字类型
      • string
      • boolean
      • object
      • null：空对象赋值：主要是和undefined进行区分。
      • undefined：变量声明不赋值的情况下，默认值是undefined；

  • 注意：在js代码中尽可能给声明的变量赋初始值。

  2. js的变量特点

    1. 变量声明只有var关键字，声明的变量可以储存任意类型的数据。
    2. js的代码可以不适用分号结尾，但是为了提升代码的阅读性，建议使用分号；
    3. js的变量允许出现同名变量，但是后面的会将前面的覆盖。
    4. 声明不赋值，默认值是undefined。

  3. js的数据类型

    1. 使用关键字typeof判断变量的数据类型。
    2. number：数值类型
    3. string：字符类型，注意：在js中字符可以使用带引号也可以使用双引号。
    4. boolean：布尔类型
    5. object：对象类型

  4. js的变量强转

    1. 使用Number()函数，将其他数据类型转换为数值类型，转换失败返回NaN（not a number）
    2. 使用Boolean()函数，将其他数据类型转换为布尔类型，有值返回true，无值返回false。

  5. 特殊的值

    1. null
    2. undefined
    3. NaN

作用：变量是用来存储数据的，方便程序进行操作
